Three ways to add CSS to HTML

  1. Inline CSS
    2. In Inline we add css direct in tag
    EX:- let you want to change the color of <h1> tag then you need to write like this
    <h1 style="color: yellow;">About me</h1>
    O/P=

    About me

  2. Internal CSS
    3. In Internal we use style tag in HTML file for style the website/text
    EX:- let you want to change the color of <h1> tag then you need to write like this
    NOTE)- Style tag is used in head section
    <style>
    body{
    background-color: green; }
    </style>
  3. External CSS
    4. In External we use .css file for style the website/text
